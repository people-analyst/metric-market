# Accept Development Velocity Card Bundle Registration

**Type:** feature | **Priority:** medium | **Status:** backlog

**Application:** metric-market

## Description

Metric Market needs to accept card bundle registration from Product Kanban for Development Velocity dashboards. Support POST /api/bundles for bundle registration and POST /api/bundles/:bundleId/data for metric snapshot pushes. Chart types needed: multi-line, stacked bar, heatmap, gauge.

## Acceptance Criteria

- [ ] POST /api/bundles accepts bundle registration from Kanban
- [ ] POST /api/bundles/:bundleId/data accepts metric snapshots
- [ ] Multi-line, stacked bar, heatmap, gauge chart types supported
